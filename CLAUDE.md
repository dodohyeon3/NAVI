# NAVI Chart — Claude 작업 규칙

## 절대 규칙: 이모티콘·특수문자 금지

파일에 이모티콘, 이모티콘 유사 기호, 장식용 특수문자를 사용하지 않는다.

금지 예시:
- 이모티콘: 🕯️ 📊 🎯 🔨 ⚖️ 📈 📉 🚀 💧 ⚡ 🧩 🚨 🎓 ✅ ❌ 🔍
- 장식 기호: ✓ ✗ ⚠ ▶ ↗ 𝚽 ● ★ ✦
- 화살표 기호: → ← ↑ ↓ (텍스트 내 장식 용도)

대신 텍스트로 표현한다:
- 완료 표시: "완료" (not "✓")
- 경고: "[주의]" (not "⚠")
- 방향: "상승" "하락" "횡보" (not "↑" "↓" "→")
- 닫기 버튼: "닫기" or "x" (not "✕")

이 규칙은 TSX, TS, JS, JSON, CSS 등 모든 파일에 적용된다.
